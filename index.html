Compras Coletivas Sapatinhos MAC Bebê

Como funciona?

Esse é um formulário do grupo de compra coletiva da marca de Sapatinhos Infantil MAC Bebê que nós mamães adoramos.

Para realizar um pedido basta escolher o modelo e preencher o formulário. Todos os modelos terão 50% de desconto + frete. O frete é de R$ 3,00 para CADA sapatino escolhido.

Pra cada modelo escolhido vocês devem preencher um formulário novo. Clique aqui para acessar o formulário.
